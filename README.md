# CBoy: Game Boy DMG Emulator

A simple Game Boy (DMG) emulator with debugger written in C++. Supports ROMs with MBC1 memory controller.

## Features
- Basic Game Boy (DMG) emulation
- MBC1 cartridge support
- Built-in debugger
- Keyboard controls

## Current Status (v0.0.1)
- ✅ Core functionality operational
- ✅ Basic graphics output
- ✅ Debugger implemented
- ⚠️ Graphical artifacts present
- ⚠️ Sound not implemented
- ⚠️ Only MBC1 supported
